### Start the Server

To run the website locally, use the provided start script:

```bash
./start.sh
```

The script will automatically detect if port 8000 is in use and stop any existing process before starting the server. Once started, open your browser and navigate to `http://localhost:8000`.

### Stop the Server

To stop the server, use the stop script:

```bash
./stop.sh
```

Alternatively, you can press `Ctrl+C` in the terminal where the server is running.
